# Print function for surveysd objects

Prints the results of a call to
[calc.stError](https://statistikat.github.io/surveysd/reference/calc.stError.md).
Shows used variables and function, number of point estiamtes as well as
properties of the results.

## Usage

``` r
# S3 method for class 'surveysd'
print(x, ...)
```

## Arguments

- x:

  an object of class `'surveysd'`

- ...:

  additonal parameters
